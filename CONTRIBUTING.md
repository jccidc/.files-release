# Contributing

Thanks for your interest in .files!

## Bug Reports

Found a bug? Open an [issue](https://github.com/jccidc/.files-release/issues) with:

- What you expected to happen
- What actually happened
- Steps to reproduce
- Your Windows version and .files version

Screenshots are always helpful.

## Feature Requests

Have an idea? Open an issue with the **feature request** label. Describe:

- What problem it solves
- How you imagine it working
- Any examples from other tools

## Code Contributions

This is a closed-source project. The source code is not available for pull requests at this time. If you have a specific fix or improvement in mind, describe it in an issue and we can discuss it.

## Questions & Support

For general questions, open a [discussion](https://github.com/jccidc/.files-release/discussions) or reach out at **jimmy@jimmycc.com**.
